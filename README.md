# EMLife
**Made by EiEddie**  
A Simple Maze Game  

## 游戏说明
- 使用方向键 `Up` `Down` `Left` `Right`控制人物
![Fge](Bin/Img/fge.png)
移动
  
- 尽可能多的收集 `Coin`
![Coin](Bin/Img/coin.png)
  
- 要想获胜, 您必须先获得地图上所有的 `Star`
![Star](Bin/Img/star.png)
  
- 当地图上显示终点
![End](Bin/Img/end.png)
时, 触碰它以获胜
  
- 当心坏蛋! 魔鬼
![Demon](Bin/Img/demon.png)
会使您输掉游戏
